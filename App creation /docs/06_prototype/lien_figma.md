## Supported Interactions

- **Screen navigation:**  
  Users can navigate between different screens by clicking on interactive buttons and UI elements.

- **Interactive UI elements:**  
  Buttons, icons, and clickable areas are defined as hotspots that trigger actions when clicked.

- **Overlay interactions:**  
  Certain elements (such as menus or dialogs) appear as overlays on top of the current screen.

- **Animated transitions:**  
  Transitions between screens include animations to visually guide the user and improve usability.
