# MVP Definition

## Purpose of the MVP

The MVP (Minimum Viable Product) aims to validate the core concept of the application:  
**allowing users to find teammates easily through simple posts and navigation**, without advanced features or backend logic.

---

## Main User Flow

1. The user lands on the main page when opening the application

2. The user navigates between the teammate search, the forum and the clips page

3. The user browses the teammate search page while aplying different filters to have a more interactive and plaisant experience

4. The user creates a team listing and while waiting for people to join him the user can access the clip or forum section 

5. If a person join the event of the user they will be put in a brief personal tchat that stops when either everybody has join and the event close or if the user decides to

6. (Optional)  The user can also modify or add elements in his profil page 

---

## Essential Screens
- **Login/register**    
    The login and register page

- **Landing Screen**  
  The landing screen is the team listing where the user can see and add team request

- **Create Post Screen**  
  Allows users to create a basic teammate request.

- **Post Detail Screen**  
  Shows detailed information about a selected teammate request or team.

- **Forum Screen**  
  Displays a list of questions where users can respond and ask other questions 

- **Profile Page**  
  Displays the users information and enable him/her to modify said information
  


  
- **Clip Page**  
  Displays multiple clips posted by other users and allow users to post clips


---

## Minimal Features

- Navigation between all essential screens

- Display of teammate listings or posts

- Ability to open and read a post

- Basic interaction to simulate creating a teammate request


---

## Explicitly Excluded from the MVP
- Backend data storage or APIs

- Moderation systems or reporting features

- Profile customization (avatars, banners, bios)
 
- Push notifications (email or in-app alerts)

---

## MVP Scope Summary

The MVP focuses strictly on **demonstrating the core idea and user flow** of finding teammates and use the main features of Gamity.
